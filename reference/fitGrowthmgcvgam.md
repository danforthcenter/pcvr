# Ease of use mgcv wrapper function for fitting gams specified by `growthSS`

Helper function generally called from
[fitGrowth](https://danforthcenter.github.io/pcvr/reference/fitGrowth.md).

## Usage

``` r
fitGrowthmgcvgam(ss, ...)
```

## Arguments

- ss:

  A list generated by `growthSS`.

- ...:

  Additional arguments passed to
  [`mgcv::gam`](https://rdrr.io/pkg/mgcv/man/gam.html).

## Value

An `gam` object, see `?gam` for details.
