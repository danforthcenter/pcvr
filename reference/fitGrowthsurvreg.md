# Ease of use wrapper function for fitting growth models specified by `growthSS`

Helper function generally called from
[fitGrowth](https://danforthcenter.github.io/pcvr/reference/fitGrowth.md).

## Usage

``` r
fitGrowthsurvreg(ss, ...)
```

## Arguments

- ss:

  A list generated by `growthSS`.

- ...:

  Additional arguments passed to
  [`survival::survreg`](https://rdrr.io/pkg/survival/man/survreg.html).

## Value

A `survreg` object.
