# Ease of use nlme wrapper function for fitting growth models specified by `growthSS`

Helper function generally called from
[fitGrowth](https://danforthcenter.github.io/pcvr/reference/fitGrowth.md).

## Usage

``` r
fitGrowthnlme(ss, ...)
```

## Arguments

- ss:

  A list generated by `growthSS`.

- ...:

  Additional arguments passed to
  [`nlme::nlmeControl`](https://rdrr.io/pkg/nlme/man/nlmeControl.html).

## Value

An `nlme` object, see `?nlme` for details.
