# Ease of use lme wrapper function for fitting gams specified by `growthSS`

Helper function generally called from
[fitGrowth](https://danforthcenter.github.io/pcvr/reference/fitGrowth.md).

## Usage

``` r
fitGrowthnlmegam(ss, ...)
```

## Arguments

- ss:

  A list generated by `growthSS`.

- ...:

  Additional arguments passed to
  [`nlme::lmeControl`](https://rdrr.io/pkg/nlme/man/lmeControl.html).

## Value

An `lme` object, see `?lme` for details.
