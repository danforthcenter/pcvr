# Ease of use nls wrapper function for fitting growth models specified by `growthSS`

Helper function generally called from
[fitGrowth](https://danforthcenter.github.io/pcvr/reference/fitGrowth.md).

## Usage

``` r
fitGrowthnls(ss, ...)
```

## Arguments

- ss:

  A list generated by `growthSS`.

- ...:

  Additional arguments passed to
  [`stats::nls`](https://rdrr.io/r/stats/nls.html).

## Value

An `nls` object, see [`?nls`](https://rdrr.io/r/stats/nls.html) for
details.
