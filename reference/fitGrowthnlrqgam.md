# Ease of use rq wrapper function for fitting gams specified by `growthSS`

Helper function generally called from
[fitGrowth](https://danforthcenter.github.io/pcvr/reference/fitGrowth.md).

## Usage

``` r
fitGrowthnlrqgam(ss, cores = getOption("mc.cores", 1), ...)
```

## Arguments

- ss:

  A list generated by `growthSS`.

- cores:

  number of cores to run in parallel

- ...:

  Additional arguments passed to
  [`quantreg::rq`](https://rdrr.io/pkg/quantreg/man/rq.html).

## Value

An `rq` object, see `?rq` for details.
