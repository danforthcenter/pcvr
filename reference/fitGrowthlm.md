# Ease of use lm wrapper function for fitting growth models specified by `mvSS`

Helper function generally called from
[fitGrowth](https://danforthcenter.github.io/pcvr/reference/fitGrowth.md).

## Usage

``` r
fitGrowthlm(ss, ...)
```

## Arguments

- ss:

  A list generated by `mvSS`.

- ...:

  Additional arguments passed to
  [`stats::lm`](https://rdrr.io/r/stats/lm.html).

## Value

An `lm` object, see [`?lm`](https://rdrr.io/r/stats/lm.html) for
details.
