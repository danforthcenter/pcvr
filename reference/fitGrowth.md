# Ease of use wrapper function for fitting various growth models specified by [growthSS](https://danforthcenter.github.io/pcvr/reference/growthSS.md)

Ease of use wrapper function for fitting various growth models specified
by
[growthSS](https://danforthcenter.github.io/pcvr/reference/growthSS.md)

## Usage

``` r
fitGrowth(ss, ...)
```

## Arguments

- ss:

  A list generated by `growthSS`.

- ...:

  Additional arguments passed to model fitting functions determined by
  `ss$type`. For type = "nlme" these are passed to
  [`nlme::nlmeControl`](https://rdrr.io/pkg/nlme/man/nlmeControl.html),
  not [`nlme::nlme`](https://rdrr.io/pkg/nlme/man/nlme.html). Additional
  arguments are documented in
  [fitGrowthbrms](https://danforthcenter.github.io/pcvr/reference/fitGrowthbrms.md),
  [fitGrowthnlme](https://danforthcenter.github.io/pcvr/reference/fitGrowthnlme.md),
  [fitGrowthnls](https://danforthcenter.github.io/pcvr/reference/fitGrowthnls.md),
  [fitGrowthnlrq](https://danforthcenter.github.io/pcvr/reference/fitGrowthnlrq.md),
  [fitGrowthmgcvgam](https://danforthcenter.github.io/pcvr/reference/fitGrowthmgcvgam.md),
  [fitGrowthsurvreg](https://danforthcenter.github.io/pcvr/reference/fitGrowthsurvreg.md),
  [fitGrowthflexsurv](https://danforthcenter.github.io/pcvr/reference/fitGrowthflexsurv.md).

## Value

A fit model from the selected type.

## See also

[growthPlot](https://danforthcenter.github.io/pcvr/reference/growthPlot.md)
for model visualization,
[testGrowth](https://danforthcenter.github.io/pcvr/reference/testGrowth.md)
for hypothesis testing,
[barg](https://danforthcenter.github.io/pcvr/reference/barg.md) for
Bayesian model reporting metrics.

## Examples

``` r
simdf <- growthSim("logistic",
  n = 20, t = 25,
  params = list("A" = c(200, 160), "B" = c(13, 11), "C" = c(3, 3.5))
)
ss <- growthSS(
  model = "logistic", form = y ~ time | group,
  df = simdf, type = "nls"
)
fitGrowth(ss)
#> Nonlinear regression model
#>   model: y ~ A[group]/(1 + exp((B[group] - time)/C[group]))
#>    data: ss[["df"]]
#>      A1      A2      B1      B2      C1      C2 
#> 204.760 148.662  13.295  10.748   3.000   3.482 
#>  residual sum-of-squares: 205524
#> 
#> Number of iterations to convergence: 4 
#> Achieved convergence tolerance: 8.288e-08
ss <- growthSS(
  model = "gam", form = y ~ time | group,
  df = simdf, type = "nls"
)
fitGrowth(ss)
#> 
#> Call:
#> stats::lm(formula = ss[["formula"]], data = ss[["df"]])
#> 
#> Coefficients:
#>      (Intercept)         bs(time)1         bs(time)2         bs(time)3  
#>            9.448           -52.515           227.337           187.069  
#>           groupb  bs(time)1:groupb  bs(time)2:groupb  bs(time)3:groupb  
#>           -2.268            69.284           -56.108           -53.586  
#> 
```
